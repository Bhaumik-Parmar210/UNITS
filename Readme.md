# Task 3: CSS Units, Box Models, Fonts

## Description
This task demonstrates the use of CSS viewport units (`vw` and `vh`) to create a responsive, centered layout without relying on Flexbox, absolute/relative positioning, or `overflow: hidden`.

## Files Included
- `index.html`: Contains the document structure, image, and button.
- `style.css`: Contains the styling rules ensuring proper box-sizing and gradient effects.
- `Readme.md`: Project documentation.

## How to View
Open `index.html` in any modern web browser.
